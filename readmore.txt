// awsIfImgOriginResponseFunction :
    - si précédente version : > supprime le trigger cloudfront de la précédente version de la fonction
    - je la modifie en direct dans l'éditeur Lambda en ligne
    - puis > "Save" > "Publish new version" > ajoute le trigger cloudfront pour cette dernière version

// awsMfImgOriginResponseFunction :
    - Lancer ces lignes de commande pour créer le paquet et l'envoyer sur Aws pour mettre à jour ma fonction lambda (forcément crée en ligne auparavant) :
        rm dist/origin-response-function.zip ;
        mkdir -p dist && cd lambda/origin-response-function && zip -FS -q -r ../../dist/origin-response-function.zip * && cd ../.. ;
        aws lambda update-function-code --function-name awsMfImgOriginResponseFunction --zip-file fileb://dist/origin-response-function.zip ;
    - si précédente version : > supprime le trigger cloudfront de la précédente version de la fonction
    - puis > "Publish new version" > ajoute le trigger cloudfront pour cette dernière version
